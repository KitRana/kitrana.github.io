Project ESAI

A contextual AI chatbot that uses NLP and other machine learning techniques to understand and respond to emergencies, helping to reduce strain on emergency response operators. A user can deploy an application which can be stand-alone or website integrated and is immediately connected to an AI operator to triage emergency situations. This system would act as a supplement to current 911 operations.

NYCity Watch 2019

NYCity Watch provides public safety information about New York City and looks at various factors including crime, EMS response times, and recent local news. The site provides a clean and modern material front-end design, interactive maps with various layers connected to API endpoints fed from NYC Open Data, data visualizations, and recent headlines scraped from popular news and social media outlets.

Visualizing Data 2019

Data can be visualized without using numbers with a visualization called a word cloud. A word cloud allows you visualize the frequency with which an item appears in a set of data or show the importance of an item in a set of data by using the font size or color of the word to indicate its weight or significance. I used word clouds to show the frequency in which a given category was associated with the movies in the dataset which consisted of movies from the TMDB Top 5000 Movies Dataset as well as elements from the OMDB API.
